# Почему из ветки feature не собирается и как исправить

## Причина

TeamCity по умолчанию запускает сборки только для веток, указанных в **VCS Root** (часто только `main` или `refs/heads/main`). Ветки вроде `feature/*` или `feature/xxx` не отслеживаются, пока их не добавить.

## Как включить сборку из ветки feature

### Вариант 1: Branch specification в VCS Root

1. В TeamCity открой **Project Settings** (или **Administration** → твой проект).
2. Перейди в **VCS Roots** → выбери корень репозитория (например `https://github.com/nolongerbob/campushelper`).
3. Найди поле **Branch specification** (или **Branches** / **Branch filter**).
4. Добавь ветки, которые должны собираться:
   - Чтобы собирать **все** ветки:
     ```
     +:*
     ```
   - Чтобы собирать только `main`, `dev` и все `feature/*`:
     ```
     +:refs/heads/main
     +:refs/heads/dev
     +:refs/heads/feature/*
     ```
   - Или просто указать конкретную ветку:
     ```
     +:refs/heads/feature/your-branch-name
     ```
5. Сохрани изменения.

### Вариант 2: Проверить Build Configuration

1. Открой **Build Configuration** (конфигурация сборки).
2. Убедись, что в **VCS Settings** или **Triggers** не задан фильтр только по `main`.
3. В **Triggers** (триггеры) обычно указано что-то вроде "VCS trigger". Проверь, что там нет ограничения по ветке (например, только `main`). Если есть — измени на нужные ветки или убери ограничение.

### Вариант 3: Запуск сборки вручную с ветки feature

1. Открой **Build** (или **Run Build**).
2. В диалоге запуска найдите поле **Branch** (или **VCS branch**).
3. Выбери или введи ветку `feature/xxx` (или ту, с которой работаешь).
4. Запусти сборку.

Если такой возможности нет — значит в VCS Root не настроены эти ветки (см. Вариант 1).

## Кратко

- **Не собирается из feature** = TeamCity не считает эту ветку “отслеживаемой”.
- **Что сделать:** в VCS Root в **Branch specification** добавить, например, `+:refs/heads/feature/*` или `+:*`, сохранить и при необходимости перезапустить сборку или дождаться следующего коммита в feature.
