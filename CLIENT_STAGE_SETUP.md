# Подключение клиента к stage-серверу

## Изменение IP-адреса сервера

Клиент теперь использует переменную окружения **`CAMPUS_HELPER_SERVER_HOST`** для адреса сервера.

### По умолчанию (localhost)

Если переменная не задана, клиент подключается к `127.0.0.1:45454` (локальный сервер).

### Подключение к stage-серверу

На машине, где запускается клиент (test-стенд), задайте переменную окружения перед запуском:

```bash
export CAMPUS_HELPER_SERVER_HOST=IP_STAGE_СЕРВЕРА
./client
```

Например, если stage-сервер на `10.211.55.5`:

```bash
export CAMPUS_HELPER_SERVER_HOST=10.211.55.5
./client
```

### Постоянная настройка

Добавьте в `~/.bashrc` или `~/.profile`:

```bash
export CAMPUS_HELPER_SERVER_HOST=10.211.55.5
```

После этого клиент всегда будет подключаться к stage-серверу.

---

**Для ЛР5 п.6:** После развёртывания на stage задайте `CAMPUS_HELPER_SERVER_HOST` на test-клиенте и запустите клиент — он подключится к stage-серверу.
